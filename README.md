ParcelTrack
===========
Track on any Malaysian courier parcel like Poslaju, CityLink, and GDex.

Features
---------
1. Trace by parcel number.
2. Keep history.
3. Arrival indicator: Red, Yellow, and Green for delivered.
4. Only support Poslaju, Citylink, and GDex temporarily.
